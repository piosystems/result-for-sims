A Form Tool for Transfer of Scores from SMC-type Scoresheet to SIMS Scoresheet

Instructions...

1. Generate the spreadsheet file from SIMS for the course (You may need to consult with Philip on this step)

2. You will notice that the file generated ends in a .xls and not .xlsx. This is a limitation. So, first open it in Excel and save as a normal .xlsx file

3. Proceed to select the above .xlsx file in form below without any modification

4. Also select in the field provided below, the equivalent SMC-type scoresheet for the course. Ensure that it contains raw values and not excel formulas in the Participation, CA1, CA2, CA3, CA4 and Exam columns, where applicable.

5. Confirm that you have selected both files above in the form and then click the Submit button

6. MAGIC: Your two files will be uploaded and almost immediately, a file will be downloaded to your system. That is your Excel sheet containing your data for upload to SIMS

7. In case SIMS cannot handle .xlsx, first open the file in Excel and save as the .xls equivalent (a downgrade)

8. Upload to SIMS (You may need to consult with Philip on this step)
